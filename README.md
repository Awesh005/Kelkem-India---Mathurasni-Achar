# Mathurasni - Authentic Indian Heritage & Flavors

![Mathurasni Banner](/legacy%20.png)

Welcome to the official digital home of **Mathurasni**, a premium brand by **Kelkem India Pvt Ltd**. We are dedicated to preserving and promoting traditional Indian recipes, bringing the authentic taste of homemade pickles, spices, and wellness products to modern households.

## 🌶️ About Mathurasni

Mathurasni represents more than just food; it's a journey through India's rich culinary heritage. Our products are handcrafted with love, using hand-picked ingredients, cold-pressed oils, and stone-ground spices to ensure every jar delivers the purity and flavor of a traditional Indian kitchen.

## ✨ Key Features

- **Premium Product Catalog**: Explore our diverse range of Pickles, Papads, Wellness supplements, and Fragrances.
- **Dynamic Filtering**: Easily navigate through categories to find your favorite flavors.
- **Visual Gallery**: A curated visual journey capturing the essence of our traditional processes and products.
- **Responsive Design**: Seamless experience across all devices—mobile, tablet, and desktop.
- **Modern Tech Stack**: Built with performance and elegance in mind.

## 🛠️ Tech Stack

- **Frontend**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)

- **Deployment**: Optimized for Cloud Run

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/mathurasni-web.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mathurasni-web
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
